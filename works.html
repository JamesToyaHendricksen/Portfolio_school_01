LifeShield AI

作品概要 必須
どんなサービス・サイトか、ターゲットユーザーは誰か

個人PCユーザー向けに、詐欺、不審メール、SNS上の異常、アカウント攻撃の兆候を分かりやすく伝える AI セキュリティサービスです。公開サイトではサービス紹介、機能紹介、料金、FAQ、お問い合わせ導線までを実装し、管理システム側は別途 Spring Boot を前提に設計しました。

使用技術 必須

フロントエンド：HTML / CSS / JavaScript

バックエンド：Java / Spring Boot / MyBatis

データベース：MySQL

その他：Git / Figma

担当範囲 必須
企画・デザイン・フロントエンド・バックエンドなど

企画、情報設計、ワイヤーフレーム作成、デザインガイドライン策定、フロントエンド実装、仕様書作成を担当しました。

工夫した点・アピールポイント 必須

個人向けセキュリティサービスとして、信頼感と親しみやすさを両立する見せ方に注力しました。LPではガラス感のあるUIや柔らかなグラデーションを用い、複雑になりやすい内容を整理して伝わる構成にしています。あわせて、管理システムの機能仕様や画面遷移まで文書化し、設計と実装の整合も意識しました。
