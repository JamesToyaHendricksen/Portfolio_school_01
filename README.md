# Portfolio Site

## 概要
個人ポートフォリオ用に制作したWebサイトです。  
自己紹介、制作物、スキル情報をまとめ、閲覧者が自分の活動内容を把握しやすい構成を目指しました。

## URL
https://jamestoyahendricksen.github.io/Portfolio_school_01/

## 使用技術
- HTML
- CSS
- JavaScript
- GitHub Pages

## 制作目的
- 自分の制作物を整理して公開するため
- Web制作の学習成果を形にするため
- 将来的に機能拡張しやすいベースを作るため

## 工夫した点
- 情報の優先順位を意識したレイアウト設計
- シンプルで視認性の高いデザイン
- GitHub Pagesで公開できるよう静的構成で作成

## 今後の展望
- レスポンシブ対応の改善
- JavaScriptを使った動きの追加
- 外部APIとの連携
- バックエンド実装を含む実用的な機能追加
